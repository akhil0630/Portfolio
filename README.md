# Portfolio
Bethi Akhil's Portfolio
Welcome to the repository for my personal developer portfolio website.

This site is built with vanilla HTML, CSS, and JavaScript, focusing on a modern, responsive design featuring dark mode, glassmorphism effects, and smooth scroll animations. It serves as a hub for my projects, experience, and technical skills.

Tech Stack
HTML5

Modern CSS3 (Grid/Flexbox)

JavaScript (IntersectionObserver API)
How to Run
Simply clone this repository and open the index.html file in any web browser.
